# MINIFLIX - Premium Netflix Clone

A high-fidelity Netflix clone built with React, Vite, and Tailwind CSS v4.

## Features
- **Cinematic UI**: Dark theme with premium glassmorphism effects.
- **Hero Section**: Dynamic featured content with "THE VIBE CODE".
- **Interactive Rows**: Smooth horizontal scrolling with hover scale transitions.
- **Simulated Auth**: Beautiful sign-in page with full navigation.

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- Lucide React
- React Router DOM

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
